# QRCodeGen

A simple Python application for generating QR codes from user-provided URLs. Built with a modern GUI using CustomTkinter, this tool allows users to input a URL and quickly create a downloadable QR code image in PNG format.

## Screenshot

![QRCodeGen Demo](Screenshot.png)

## Features
- **User-Friendly Interface**: Clean, themed GUI that adapts to system appearance (light/dark mode).
- **Easy QR Generation**: Enter a URL and generate a QR code with one click.
- **Customizable Output**: Generates high-quality PNG images with configurable error correction and sizing.
- **Error Handling**: Validates input and provides feedback for empty or invalid entries.
- **Lightweight**: Minimal dependencies for fast setup and execution.

## Requirements
- Python 3.8 or higher
- Dependencies: `qrcode[pil]`, `customtkinter`
